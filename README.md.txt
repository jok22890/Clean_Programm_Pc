# FileDeleter 🧹

Утилита на C++ для быстрого и массового удаления файлов определенного расширения в заданной директории. 
Измеряет время работы с помощью встроенного таймера.

## Функционал
- Поиск файлов по указанному пути.
- Фильтрация файлов по расширению (регистронезависимая).
- Замер точного времени, потраченного на удаление.

## Требования
- Компилятор с поддержкой **C++17** или выше (используется `std::filesystem`).
- ОС Windows (для корректной кодировки консоли).
